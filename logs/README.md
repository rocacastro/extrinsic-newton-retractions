# Logs

- `reference_generation_2026-07-16.log` records the controlled execution that produced the committed reference tables.
- `supplement_compilation.log` records the final LaTeX compilation of the committed supplementary PDF.
- `repository_validation.log` records the final repository-consistency checks.

Local execution logs may be stored in this directory using names such as `local_generation.log`. Files matching `logs/local_*.log` are ignored by Git.
